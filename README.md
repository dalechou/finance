# finance

This is a a basic stock and forex scraping tool that is probably good enough for personal use. 


# How to use this tool?

ticker.txt can be edited to add stock ticker symbol. 

forex.txt can be edited to add forex pairs.

The tool uses Alpha Advantage API key, which has a free tier. Add it as a repository secret as 'ALPHA_VANTAGE_API_KEY'.

You will probably need to set up a PAT. Add it as a repository secret as 'PERSONAL_ACCESS_TOKEN'.
